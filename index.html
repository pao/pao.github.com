    Julia

    The Julia language is a fast, JIT-compiled, dynamic language designed for the needs of scientific and numerical computing but made to be usable as a general-purpose language. (GitHub, my fork)

    Monads.jl (docs)

    Starting from a solution to the final problem in Seven Languages in Seven Weeks, an implementation of several common monads as well as a macro which implements syntax similar to Haskell's do notation.

    StrPack.jl (docs)

    The binary data stream Swiss army knife. Chop streams into structures, or put structures into streams, while keeping to arbitrarily-defined requirements for alignment and insertion of padding.

    QuickCheck.jl (docs)

    Julia's implementation of Haskell's famous QuickCheck specification-based randomized testing framework.

    Experiments

    CreatureBattle

    (MATLAB, Haskell, Python) The CreatureBattle system was originally determined to settle which of two hybrid animals would win in a fight (i.e., crocopottomocopter vs. Siberian gorilloctoshark), but it's generic enough to settle disputes about any two arbitrary things which are expressible as strings. Besides the canonical MATLAB version, there are also interoperable Python and Haskell implementations.

    Re2.jl

    (incomplete) My attempt to create an alternative to Julia's standard PCRE-based regex engine using the Google re2 DFA-based engine and the cre2 C API. Very unstable.

    Gists

    CyanogenMOD EC2 development environment setup

    (Shell) Scripts for initializing and managing CyanogenMOD development environments on Amazon EC2 instances. May not work anymore; last used for CM7.

    Emacs Client Smartframe

    (Shell) Script which can be used as EDITOR, automatically starts emacsclient if needed, but never opens a new frame if one is available. Also provides a shell alias edit which is useful if you like Emacs, but still want to mostly work from a traditional shell.

    getHeart

    (Python) Batch downloader of digital materials for participants in Jonathan Coulton's Artificial Heart Level 4.

    Jo(de)Co(der)

    (Python) A script which decodes the easter-egg image in the Artificial Heart Level 4 Javascript "game." The base-64 encoded image appeared to be accidentally mangled in the process of (hilariously) obfuscating the rest of the program. This script solves for the replaced strings and outputs the original PNG.

    Linux kernel module template

    (C) Tiny template for a Linux kernel module.

    MLP Music Archive update script helper

    (Python) Compares two directories and outputs both a Windows batch and Unix shell script which deletes files and paths that don't exist in the new directory. Used for a couple of releases.

    Just for fun

    Cards Against Equinity (my fork)

    (Python, Javascript) A live online version of Cards Against Humanity played over websockets all running in Twisted Python. My primary contribution was a drastic simplification of the backend, consolidating everything onto Twisted and using flat files for the modest amount of data. Play live at http://cah.berrytube.tv/!
